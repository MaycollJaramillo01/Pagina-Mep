¿Qué ofrecemos?

En nuestros talleres de software aprenderás las tecnologías más demandadas, desde desarrollo web hasta programación avanzada. Únete a una comunidad de entusiastas de la tecnología y mejora tus habilidades para destacar en el mundo digital.

Nuestros Talleres

Desarrollo Web

En este taller aprenderás a construir páginas web responsivas usando HTML, CSS y JavaScript. Ideal para quienes desean iniciar en el mundo del desarrollo web.

Programación en Python

Aprende a programar en Python, uno de los lenguajes de programación más versátiles y utilizados en la actualidad. Desde conceptos básicos hasta el desarrollo de aplicaciones avanzadas.

Desarrollo de Aplicaciones Móviles

Domina el desarrollo de aplicaciones móviles con Android e iOS. Este taller es perfecto para aquellos que buscan crear aplicaciones que impacten el mercado móvil.
